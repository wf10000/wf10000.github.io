领导者的核心就是通过营造氛围提升绩效

管理者角色分为三层：

  • 领导者
  • 管理者
  • 执行者

案例：俄罗斯去旅行，看到俄罗斯火车站里面两个人，两个人拿着铁锹，一个人挖坑，另外一个人填上，一路挖坑填上。上前询问，回答，我们在种树，我的任务是挖坑，另外一个是填坑，问没看见你们种树啊，答种树的那个人请假了。

我只干我的事情，这不是执行。

执行的定义：给出结果

初级管理者以执行为重
中级管理者每一层都要做好
高级管理者以氛围为重

如果你把工作视为刻意练习的时候，你会发现其中的乐趣。

一个普通员工怎么成为一个管理者

  • 1.建立信任（执行者的层面）

  • 2.建立团队（管理者的层面）

    误区：并不是我干的活越多，干的越好，团队的人越服我。

    • 怎么让大家服你:一定要学会放手。

  • 3.建立体系（管理者的层面）

    • 输入和输出的比例决定着体系的好坏
    • 好的体系只要普通的团队就能做出一流的东西

关于创业的概念推荐一本书《突破瓶颈》
