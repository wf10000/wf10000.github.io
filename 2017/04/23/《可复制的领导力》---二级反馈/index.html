《可复制的领导力》（9）---二级反馈

发表于 2017-04-23 | 更新于 2017-04-23 | 分类于 读书 | | 阅读次数
字数统计

二级反馈可以有效的揭示对方的盲点象限
二级反馈的前提是真诚，善于发现他人的优点是感恩的表现

二级反馈的难点

  • 1.你要不断的发现别人身上的优点
    案例：陶行知的三颗糖
    幸福完全来自于你的价值观
  • 2.要说清楚为什么
    怎么教小孩
  • 传统的做法：贿赂、控制、放任
  • 正确的做法：情感引导

推荐一本书《你就是孩子最好的玩具》
自尊水平下降导致的问题就是要依靠他律，不能自律

  • 3.要避免说“但是”
    表扬完加一个但是是错误的做法，不要把二级反馈和负面反馈放在一起。

二级反馈的流程：

  • 寻找好的东西
  • 建立自信
  • 设定标准
  • 建立信任
  • 告诉他们为什么，是什么

怎么批评

BIC（behaver has inspect which leads to consequence）
你的某一个行为产生了什么影响从而导致了什么后果

错误行为两种表达方式：

要把你要表达的观点停住，改成事实

  • 观点
    小张，你最近经常迟到。
  • 事实
    小张，你上周有3次迟到。

影响 和 后果

影响：短期的局部的就能看到的负面结果
要和他自己的利益挂钩，让他感觉到你是为了帮他
案例：这样下去会导致我们的班风下降，也不利于你的形象。
