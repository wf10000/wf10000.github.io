《可复制的领导力》（2）---如何处理员工关系

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

员工有三个类型

  • 自燃
  • 点燃
  • 阻燃

    如何让员工主动工作

  • 四年之后你希望自己成为什么样的人
  • 与员工一起制定目标，签订口头协议，明确公司和个人应尽的义务和负担的责任。

    世界状态分布的两种形式

  • 正态分布
  • 幂次分布

    把公司比喻成大家庭

    会引发更多的矛盾
  • 你不会开掉你儿子
  • 如果有任何人走掉，大家都会月的

    公司应该是一个球队

    个人的目标和公司的目标一致

    要把员工视作公司的投资人

    你要为投资人的投资负责
    让员工2年后离开公司的时候，价值更高

    领导一定要把控好方向

    要区分关键要务，不要把所有问题一把抓，每一段时间重点解决一个关键要务。

    三种人

  • prisonner
    挑战我执，才能不让自己的偏见束缚自己
  • passenger
    你讲的好，我就听你过一会。你讲的不好，我就眯一会。
  • explorer
    保持不断学习

有些人一上班就是prisonner,一回家就是explorer或者passenger。

你要成为哪种人，取决与你自己。人最难的法门就是反观自己。
