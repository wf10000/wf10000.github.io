《可复制的领导力》（7）---如何倾听

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

只有善于倾听，才能当领导

倾听传递的最重要的是尊重和信任，领导力就是尊重和信任

倾听的流程

  • 1.深呼吸
    西游记是一个人修炼的过程：孙悟空代表不被约束的心，猪八戒代表欲望，沙僧代表理性和逻辑，白龙马代表意志。
  • 2.提问
    开放式。
  • 3.倾听与回应
    点头、微笑
  • 4.复述
  • 5.总结

一个人撒谎的时候会摸自己的自我安慰区。女人的自我安慰去是胸口，男人的自我安慰区是后脑勺。

骂人的资本：你跟他之间建立了情感账户

如何面对对方情绪失控的情况

反映情感（ 反应对方的情绪-准确的陈述对方此刻的情绪和产生的原因）
举例：

  • 关于此事你看起来好像很心烦
  • 是不是刚才我那样说让你很生气
  • 我能够理解你，真的压力很大

小技巧：当你压不住火的时候，只要点头，你的情绪水平就回立刻下降。
导入效应：我们平时总以为我们的思想影响了我们的行为，其实我们的行为也会反过来影响我们的思想。

案例：销售在推销产品的时候问的所有问题，都是要你回答yes的。然后慢慢你就被套进去了。
