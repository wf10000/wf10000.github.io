AndroidStudio快捷键

发表于 2017-03-22 | 分类于 Tools | | 阅读次数

根据AndroidStudio的官方快捷键说明清单整理了一份快捷键对照表

通用

功能描述 快捷键
自动弹出可以补全当前变量名或者方法名的候选列表 Ctrl + Shift + Space
全局搜索(注意不要被输入法拦截) Double Shift
列出当前可能的动作，便于快捷修复逻辑（比如快捷创建函数、为当前语句插入if判断） Alt + Enter
快捷插入构造函数、getter、setter等函数 Alt + Ins
显示当前变量的声明信息 Ctrl + P
选择当前光标对应完整的命名，多次按此组合会依次扩大选中区域 Ctrl + W
依次缩小选中区域 Ctrl + Shift + W
显示最近打开的文件列表 Ctrl + E
重命名 Shift + F6
打开对应的窗口(例如Alt+1打开project窗口) Alt + [0-9]
同步(检查所有通过外部改变的文件，重新加载) Ctrl + Alt + Y
最大化编辑窗口 Ctrl + Shift + F12
代码检查 Alt + Shift + I
快速切换主题 Ctrl + BackQuote (`)
打开设置 Ctrl + Alt + S
打开工程设置 Ctrl + Alt + Shift + S
查找快捷功能 Ctrl + Shift + A
插入预设的模版 Ctrl + J
