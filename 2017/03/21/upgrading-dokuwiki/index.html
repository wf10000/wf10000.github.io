升级DokuWiki

发表于 2017-03-21 | 分类于 Web | | 阅读次数

DokuWiki升级起来很简单，只是把新版本的目录直接覆盖旧版本即可。可以参考以下简单的步骤

先备份你的DokuWiki

这一步不是必须的，但是强烈建议先备份。

阅读更新说明

仔细阅读版本 发布日志 说明，看看有没有特别注意的地方

下载最新版本

更新

解压并覆盖原来的版本。注意一定要保证所有文件都要覆盖。

检查权限

检查覆盖的目录和文件的权限

  • data/ 、 data/tmp/ ：www-data权限
  • lib/plugins/ : www-data权限
  • lib/ : 755权限
  • lib/tpl ：www-data权限
  • conf/ : www-data权限

删除没用的文件

在data/delete.list路径下找到这个文件，里面列出了新版本移除的文件列表。需要依次检查，如果本地仍然存在列表中的文件，则手动删除。

更新插件

检查有没有可更新的插件，根据自己的需要选择更新
