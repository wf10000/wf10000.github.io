Window系统下dokuwiki安装指引

发表于 2017-04-05 | 分类于 工具 | | 阅读次数

下载安装xampp

  • 下载地址: https://www.apachefriends.org/zh_cn/index.html
  • 安装位置不要默认装在Program File文件夹下，要自己创建一个没有任何空格或符号的文件夹。例如d:\xampp

下载安装dokuwiki

  • 下载地址:https://download.dokuwiki.org/
  • 压缩包解压出来的目录放到xampp的apache的发布目录下，我的目录是d:\xampp\htdocs

配置

  • 启动xampp-control，启动apache，如下图
  • http://127.0.0.1/dokuwiki/install.php完成dokuwiki的初始化配置。
  • 初始化配置完成可以访问http://127.0.0.1/dokuwiki/doku.php进入douwiki页面

给dokuwiki添加插件

  • 下载插件文件，解压之后，放到d:\xampp\htdocs\dokuwiki\lib\plugins目录中就可以。然后在管理登陆的页面中就会提示插件安装成功。

《可复制的领导力》---什么是领导力

发表于 2017-04-04 | 分类于 读书 | | 阅读次数

领导力的核心是一系列工具

  • 怎么表扬员工
  • 怎么批评员工
  • 教员工怎么创新
  • 怎么授权

交代任务要反复确认，确保对方理解

  • 日本人交代一个任务要重复5遍

现在员工为什么这么难管

  • 互联网放大了每个人的个人能力

领导力发生了新变化，要用游戏化的方式进行组织管理

游戏化组织需具备四个特征

  • 宏大的目标
    • 目标要清晰、具体、与社会有关
    • 推荐相关书籍：《指数型组织》，《哈佛商学院的最受欢迎的领导课》
    • 愿景是所有人都认可并愿意为之付出努力的目标
  • 规则
    • 清晰明确的规则：海星模式—韩都衣舍、airbnb、优步
  • 及时反馈
    • 排名是一种典型的反馈—打飞机游戏
    • 人类很重视别人对自己的评价，所以不要吝啬对别人的反馈。
  • 自愿参与
    • 推荐一本书《联盟》

DokuWiki升级起来很简单，只是把新版本的目录直接覆盖旧版本即可。可以参考以下简单的步骤

先备份你的DokuWiki

这一步不是必须的，但是强烈建议先备份。

阅读更新说明

仔细阅读版本 发布日志 说明，看看有没有特别注意的地方

下载最新版本

更新

解压并覆盖原来的版本。注意一定要保证所有文件都要覆盖。

检查权限

检查覆盖的目录和文件的权限

  • data/ 、 data/tmp/ ：www-data权限
  • lib/plugins/ : www-data权限
  • lib/ : 755权限
  • lib/tpl ：www-data权限
  • conf/ : www-data权限
