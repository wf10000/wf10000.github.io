《可复制的领导力》（6）---设定目标

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

为什么设定目标

  • 打造一致性，将员工的目标与经理的目标统一
  • 为绩效的回顾提供依据
  • 为工作的调整和学习的提升提供依据
  • 为个人的满意和赞誉提供基准

寻找亮点

当我们感觉到做某件事资源不够的时候，解决它一种突破性创新思维是寻找亮点

目标分三类

  • 方向性目标
  • 过程性目标
  • 理性目标

目标设定流程

  • 1.沟通
  • 2.设定
  • 3.一致
  • 4.跟进（职位越低的人跟进的频率越高）

SMART原则

  • 1.具体的（specific）
  • 2.可衡量的（Measurable）
  • 3.可实现（Achievable）
  • 4.与工作相关的（Realistic）
  • 5.时间要求（Time Bound）

目标书写公式

  • 1.动词
  • 2.任务
  • 3.衡量的方法
  • 4.目标

    示例：展现标杆服务，在07年10月31日前为至少2位公司客户完成制定标杆报告，确保遵守内部要求和法律规定，在2007年6月的核查中100%合规

个人管理模型

有时候你会觉得心烦，不知道该从哪里着手

  • 1.目标：有了一个业绩，和一个团队，反推出所有的目标。
  • 2.条件：把每一个目标的阻碍列出来。再写下帮助的条件。
  • 3.个人特性：再列出自己个人特性—对于担当好工作团队的领导角色以及达成重要目标最为重要的技能经验和特性。
  • 4.角色：再按领导、管理、执行分别列出你接下来要做的事情
    有时候觉得自己脑子乱，纯粹是因为自己的情绪，以上的方法就是帮助克服情绪

谁害死了这条鱼

当你被别人推理的时候，你会觉得很受伤害。

推理阶梯

  1. 观察数据
  2. 选择数据

    案例：你看着越顺眼的人就越顺眼，你看着越讨厌的人就越讨厌

    • 推荐一本书丹尼尔-卡尼曼的《思考快与慢》
      案例：一个教授做了一个实验，让同学们看一个视频，两个人传球，数传了多少次，传完之后问大家有没有看到一只大猩猩，大家都说没看到，重新播放，发现有一个大猩猩很明显在视频里。
    • 没有人是客观的
  3. 给数据以意义
  4. 做出假设
  5. 得出结论
  6. 采纳自己的观点
  7. 采取行动

在你做一个决定之前，问自己：这有没有可能只是我的一个推理

戒除推理的方法：沟通

不要轻易推理别人，也要减少被别人推理的可能性

  • 主动沟通
  • 约束好自己

《可复制的领导力》（5）---如何营造氛围

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

领导者的核心就是通过营造氛围提升绩效

管理者角色分为三层：

  • 领导者
  • 管理者
  • 执行者

案例：俄罗斯去旅行，看到俄罗斯火车站里面两个人，两个人拿着铁锹，一个人挖坑，另外一个人填上，一路挖坑填上。上前询问，回答，我们在种树，我的任务是挖坑，另外一个是填坑，问没看见你们种树啊，答种树的那个人请假了。

我只干我的事情，这不是执行。

执行的定义：给出结果

初级管理者以执行为重
中级管理者每一层都要做好
高级管理者以氛围为重

如果你把工作视为刻意练习的时候，你会发现其中的乐趣。

一个普通员工怎么成为一个管理者

  • 1.建立信任（执行者的层面）

  • 2.建立团队（管理者的层面）

    误区：并不是我干的活越多，干的越好，团队的人越服我。

    • 怎么让大家服你:一定要学会放手。

  • 3.建立体系（管理者的层面）

    • 输入和输出的比例决定着体系的好坏
    • 好的体系只要普通的团队就能做出一流的东西

关于创业的概念推荐一本书《突破瓶颈》

  • 4.建立文化（领导者的层面）

《可复制的领导力》（7）---如何倾听

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

只有善于倾听，才能当领导

倾听传递的最重要的是尊重和信任，领导力就是尊重和信任

倾听的流程

  • 1.深呼吸
    西游记是一个人修炼的过程：孙悟空代表不被约束的心，猪八戒代表欲望，沙僧代表理性和逻辑，白龙马代表意志。
  • 2.提问
    开放式。
  • 3.倾听与回应
    点头、微笑
  • 4.复述
  • 5.总结

一个人撒谎的时候会摸自己的自我安慰区。女人的自我安慰去是胸口，男人的自我安慰区是后脑勺。

骂人的资本：你跟他之间建立了情感账户

如何面对对方情绪失控的情况

反映情感（ 反应对方的情绪-准确的陈述对方此刻的情绪和产生的原因）
举例：

  • 关于此事你看起来好像很心烦
  • 是不是刚才我那样说让你很生气
  • 我能够理解你，真的压力很大

小技巧：当你压不住火的时候，只要点头，你的情绪水平就回立刻下降。
导入效应：我们平时总以为我们的思想影响了我们的行为，其实我们的行为也会反过来影响我们的思想。

案例：销售在推销产品的时候问的所有问题，都是要你回答yes的。然后慢慢你就被套进去了。

员工有三个类型

  • 点燃
  • 阻燃

    如何让员工主动工作

  • 四年之后你希望自己成为什么样的人
  • 与员工一起制定目标，签订口头协议，明确公司和个人应尽的义务和负担的责任。

    世界状态分布的两种形式

  • 正态分布
  • 幂次分布

    把公司比喻成大家庭

    会引发更多的矛盾
  • 你不会开掉你儿子
  • 如果有任何人走掉，大家都会月的

    公司应该是一个球队

    个人的目标和公司的目标一致

    要把员工视作公司的投资人

    你要为投资人的投资负责
    让员工2年后离开公司的时候，价值更高

    领导一定要把控好方向

    要区分关键要务，不要把所有问题一把抓，每一段时间重点解决一个关键要务。

    三种人

  • prisonner
    挑战我执，才能不让自己的偏见束缚自己
  • passenger
    你讲的好，我就听你过一会。你讲的不好，我就眯一会。
  • explorer
    保持不断学习

有些人一上班就是prisonner,一回家就是explorer或者passenger。

你要成为哪种人，取决与你自己。人最难的法门就是反观自己。

管理者的定义

通过别人来完成工作

  • 不懂得管理者定义的人会有什么问题：
    1. 员工觉得你不信任他
    2. 累死自己

案例：诸葛亮
所有事情都亲力亲为，分派锦囊，功劳都是军师的。
挥泪斩马谡—不允许员工犯错
蜀中无大将，廖化作先锋

  • 眼睁睁看着员工犯错，就是不说。
  • 管理者最重要的责任是：培养他人的成长

时间管理

分为4个象限：重要紧急、重要不紧急、紧急不重要、不紧急不重要

  • 如果你一直优先做重要紧急的事，你会永远都在做重要紧急的事。
  • 一件重要不紧急的事不做会产生十件重要紧急的事
  • 要容忍员工有自己的风格，前提是他对这件事很认真。

要优先做重要不紧急的事情

  • 锻炼身体
  • 陪孩子
  • 人生规划

时间管理的最高境界

至人之用心若镜，物来则现，物去不留

时间管理第二代理论

getting things down GTD

如何人生均衡发展

每天要做的八件事

  • 事业
  • 健康
  • 家庭
  • 理财
  • 人脉
  • 学习
  • 休闲
  • 心灵
    • 简单的就每天花十分钟静坐

有带团队能力的人最值钱

  1. 只有你的员工成长了，你的工作才能轻松，你才能有更多的时间迎接更多的机会和挑战。
  2. 管理者亲自要做的几件事：
  • 计划
  • 组织
  • 协调
  • 监督

作为领导，营造氛围很重要

营造氛围的情况：

  • 关爱员工
  • 定期请员工吃个饭，聊聊天
  • 有事替员工背锅
  • 破坏氛围的情况
  • 出尔反尔
  • 当着员工的面说丧气的话

沟通的视窗（乔哈里视窗）

是一种关于沟通的技巧和理论，页被称为“自我一世的发现—反馈模型”

公开象限

自己知道 且 他人知道

公开象限大的好处：

1. 有影响力
2. 别人的尊重
3. 容易得到别人的信任

公开象限大的坏处：

隐私象限

自己知道 他人不知

隐私有三类：

  • dds （deep dark secret）
    1. dds太多会有焦虑症，压力太大
    2. 书有未曾经我读，事无不可对人言
  • 不好意思说
    案例：
    1. 暗恋
      花堪折时直须折，莫待无花空折枝
    2. 自己的意愿不愿意给领导说

  • 忘了说

    以为别人知道—知识的诅咒

    1. 一个员工离职可能一个原因是：永远不会当面夸员工，更多的只是指责。
    2. 所有的营销工作都是为了打破知识诅咒
    3. 农夫山泉有点甜
    4. 一个人一旦对某一专业太熟，就很容易陷入知识诅咒
    5. 故事是打败知识诅咒最有效的
    6. 关于打破知识诅咒，推荐书籍—《让创意更有粘性》

      盲点象限

      自己不知道，别人知道

要点：

1. 神经比较大条的人：他把自己身边的人都得罪完了，但是他自己一点感觉都没有。还夸自己：我这个人最大有优点就是直来直去，大家都喜欢我。
2. 盲点象限大是很危险的。
3. 你要相信自己的缺点很难被自己发现。

如何解决：

  • 闻过则喜—大禹
  • 闻过则拜—子路
  • 闻过则问

我们该找谁来分享我们的盲点
案例：邹忌讽齐王纳谏
老婆-爱我，小妾-怕我，门客-有求于我
投诉是发现盲点的有效途径

潜能象限

自己不知道，别人也不知道

案例：力克胡哲

成长的过程是公开象限不断放大的过程。
如何主动放大公开象限

  • 把隐私象限转化为公开象限—自我揭示
    1. 跟别人讲讲自己的人生经验
  • 把盲点象限转化为公开象限—恳请反馈

这个理论同样可以应用到家庭中，要尽力扩大公开象限。

《可复制的领导力》（8）---给予和接受反馈

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

正确的给予反馈是领导力的分水岭

意义

管理者的责任是帮助员工去达标，而不是考评惩罚
让员工从要我做变成我要做

反馈的方式

  • 1.积极性、鼓励性反馈
  • 2.纠正式、调整式反馈
    反馈 ！= 表扬/批评
    表扬和批评是已经得出了结论，反馈是一种事实的提出

    反馈分三级

  • 零级反馈：员工做对了一件事，你看到了，但是你没有肯定
    零级反馈导致的后果：职业倦怠感、家庭倦怠感
  • 一级反馈：表扬、提拔
    喜欢吵架是因为自己没有安全感
  • 二级反馈：给予赞扬，为什么，是什么
    让员工感受到这件事情的意义（塑造员工的行为的有效方法）

做对事是塑造行为的最好机会
做错事是拉近距离的最好机会

案例：

推荐一本书《一分钟经理人》

自尊带来自律
无助带来叛逆
生存的本能是只能看到别人的缺点

金钱的弱点

  • 衰减
    今年发1万，明年也发1万的效果就回衰减
  • 腐蚀
    会让这件事丧失它原来的意义

《可复制的领导力》（10）---负面反馈

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

三明治沟通法：然而、但是、虽然
要避免三明治沟通法。

绩效谈话的步骤

  • 一、准备
    准备事实、影响、后果
  • 二、谈话
    • 1.设定情景。“这次来找你是谈谈XXX的问题”，不要用负面的词汇
    • 2.给予反馈。把BIC一口气说完。不要过早的把话语权交给员工。
    • 3.倾听和鼓励。同理心倾听。鼓励是让他重新找回自信心。一个人自信的时候才愿意面对自己的问题。
    • 4.商讨改变。把谈话成就感交给员工。
  • 三、行动总结
    落实到具体的行动
  • 四、跟进计划

一个优秀的团队，二级反馈：负面反馈=4：1

调整心态：把工作视为一种练习
什么样的人进步快：经常发现自己进步的人

《可复制的领导力》（9）---二级反馈

发表于 2017-04-23 | 分类于 读书 | | 阅读次数

二级反馈可以有效的揭示对方的盲点象限
二级反馈的前提是真诚，善于发现他人的优点是感恩的表现

二级反馈的难点

  • 1.你要不断的发现别人身上的优点
    案例：陶行知的三颗糖
    幸福完全来自于你的价值观
  • 2.要说清楚为什么
    怎么教小孩
  • 传统的做法：贿赂、控制、放任
  • 正确的做法：情感引导

推荐一本书《你就是孩子最好的玩具》
自尊水平下降导致的问题就是要依靠他律，不能自律

  • 3.要避免说“但是”
    表扬完加一个但是是错误的做法，不要把二级反馈和负面反馈放在一起。

二级反馈的流程：

  • 寻找好的东西
  • 建立自信
  • 设定标准
  • 建立信任
  • 告诉他们为什么，是什么

怎么批评

BIC（behaver has inspect which leads to consequence）
你的某一个行为产生了什么影响从而导致了什么后果

错误行为两种表达方式：

要把你要表达的观点停住，改成事实

  • 观点
    小张，你最近经常迟到。
  • 事实
    小张，你上周有3次迟到。

影响 和 后果

影响：短期的局部的就能看到的负面结果
要和他自己的利益挂钩，让他感觉到你是为了帮他
案例：这样下去会导致我们的班风下降，也不利于你的形象。
